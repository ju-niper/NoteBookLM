# NotebookLM Frontend

React + TypeScript + Tailwind v3 UI for the NotebookLM clone.

## Main UI Features

- Source panel (upload files + web search/import)
- Chat panel with:
  - session overview section (title, summary, suggested questions)
  - streaming answers
  - inline citation badges with popovers
  - follow-up questions
- Source viewer panel (raw source content + citation scroll target)

## Run

```bash
npm install
npm run dev
```

Frontend default URL: `http://localhost:5173`
